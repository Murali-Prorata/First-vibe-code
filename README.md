# Webpage with Ad

A simple HTML webpage featuring a centered advertisement box.

## Features

- Clean, centered layout
- Ad box with headline text
- Two clickable links
- Responsive design
- Hover effects on links

## Files

- `index.html` - The main webpage file

## How to Use

1. Open `index.html` in your web browser
2. View the ad with the headline and links
3. Click on "link 1" or "Link 2" to navigate (currently set to placeholder links)

## Customization

- Edit the headline text by changing "headline" in the `<h1>` tag
- Update link text and URLs by modifying the `<a>` tags
- Adjust styles in the `<style>` section to customize colors, spacing, and layout

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge)
